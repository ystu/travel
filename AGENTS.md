# 旅遊安排 Guideline

> 目的：在決定行程前，先用同一套標準檢查天氣、住宿 CP 值、素食選擇與交通可行性，避免只看景點清單而忽略實際體感與移動成本。

## 1. 天氣觀察

- [ ] 查詢旅行期間每日天氣：溫度、降雨機率、風速、紫外線、體感溫度。
- [ ] 觀察目的地季節特性：梅雨、颱風、東北季風、濃霧、海象、潮汐。
- [ ] 把戶外景點排在天氣較穩定的日期，室內景點留作雨備。
- [ ] 若有船班、海上活動、登山或離島行程，出發前 3 天與前 1 天再次確認。
- [ ] 每天安排一個彈性時段，方便因天氣調整順序。

### 天氣判斷原則

- 降雨機率高：減少海邊、步道、長時間騎車或拍照型景點。
- 風速強：避免高曝曬海岸、跨海大橋、海上活動與長距離機車移動。
- 高溫或高紫外線：中午安排午餐、咖啡、博物館、住宿休息。
- 天氣不穩：優先保留可取消或可改期的活動。

## 2. 住宿 CP 值

- [ ] 先確認主要活動區域，再決定住宿位置，避免省房價卻增加交通成本。
- [ ] 比較每晚價格、房型、評分、近期評論、取消政策與附加服務。
- [ ] 確認房型是否有對外窗，並優先選擇較高樓層。
- [ ] 檢查是否含早餐、停車位、接駁、洗衣、寄放行李、租車合作。
- [ ] 看近期負評重點：隔音、清潔、冷氣、熱水、床況、交通便利性。
- [ ] 計算總成本：房價 + 交通 + 停車 + 早餐 + 時間成本。

### CP 值判斷原則

- 高 CP 值不等於最低價，而是「位置、品質、彈性、總成本」平衡。
- 若每天都早出晚歸，可以降低房內設施權重，提高位置與停車權重。
- 若重視採光、通風與安靜度，對外窗與高樓層要列為訂房備註，不只看房型名稱。
- 若同行有長輩、小孩或素食者，住宿附近的餐食與便利性要加權。
- 若遇連假或旺季，取消政策與備案住宿要比平日更重要。

## 3. 素食安排

- [ ] 先標出每天午餐、晚餐附近可用的素食店或友善餐廳。
- [ ] 確認餐廳類型：全素、蛋奶素、五辛素、可客製素食。
- [ ] 查營業時間與公休日，特別注意下午空班與晚餐提早打烊。
- [ ] 每餐至少準備 1 個首選與 1 個備案。
- [ ] 若前往離島、郊區或景點密集區，先準備麵包、水果、飯糰、零食。

### 素食判斷原則

- 景點附近只有小吃時，要提前看是否有素食主食，不只看飲料或點心。
- Google Maps 評論中搜尋「素」「vegetarian」「vegan」「蛋奶」等關鍵字。
- 若餐廳不是素食店，出發前可電話確認是否能做素食。
- 重要餐點不要排得太晚，避免店家賣完或休息。

## 4. 交通規劃

- [ ] 確認主要交通方式：自駕、機車、公車、包車、步行、船班、接駁。
- [ ] 每段移動估算實際時間，並加上停車、等車、買票與找路緩衝。
- [ ] 檢查景點間是否順路，避免來回折返。
- [ ] 對機場、車站、港口、船班、退房時間設定硬性時間點。
- [ ] 每天最後一段路線避免排太滿，預留延誤與體力下降空間。

### 交通判斷原則

- 離島或海線行程要優先確認船班與天候，不要只看地圖距離。
- 機車適合短距離彈性移動，但要把風、雨、曝曬與行李納入考量。
- 自駕要檢查停車難度、道路寬度、尖峰時段與夜間駕駛風險。
- 公車行程要先查末班車與班距，避免只查單趟路線。

## 5. 每日行程檢查

- [ ] 每天景點數量是否合理。
- [ ] 午餐與晚餐是否都有素食可行方案。
- [ ] 戶外景點是否避開最熱、風最大或雨勢最高的時段。
- [ ] 是否有至少一段可刪減的彈性行程。
- [ ] 重要交通節點是否已提前 30 至 60 分鐘緩衝。
- [ ] 若天氣變差，是否能改成室內、短程或市區路線。

## 6. 出發前確認節奏

- 出發前 7 天：確認住宿、交通票券、租車、主要餐廳與高需求活動。
- 出發前 3 天：重新檢查天氣、船班、營業時間與餐廳公休日。
- 出發前 1 天：確認隔天路線、集合時間、票券、現金、雨具與備案。
- 每天晚上：依隔天天氣與體力調整景點順序。
