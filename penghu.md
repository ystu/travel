# 澎湖四天三夜行程

> 地點：澎湖  
> 天數：四天三夜  
> 住宿：馬公市區連住 3 晚  
> 交通：自駕開車  
> 飲食：蛋奶素  

---

## Day 1｜馬公市區＋南環海景

- [ ] 09:00 - 11:00 抵達澎湖、租車、前往住宿放行李
- [ ] 11:30 - 12:30 午餐：蔡記饡蔬食坊
- [ ] 13:00 - 14:00 篤行十村
- [ ] 14:15 - 15:15 中央老街、天后宮
- [ ] 15:30 - 16:30 觀音亭、彩虹橋
- [ ] 17:00 - 18:00 風櫃洞
- [ ] 18:15 - 19:00 山水沙灘
- [ ] 19:30 - 21:00 晚餐：回市區找蔬食餐廳

**備註**
- 第一天以市區和南環輕鬆走為主
- 蛋奶素可優先選早餐店、咖啡店、蔬食店的蛋奶品項
- 馬公市區晚上較熱鬧，建議停外圍再步行進中央老街、天后宮一帶

---

## Day 2｜北環一日遊

- [ ] 08:00 - 09:00 早餐：舒心食光或先買早餐
- [ ] 09:15 - 09:40 中屯風車
- [ ] 10:00 - 10:30 通梁古榕
- [ ] 10:40 - 11:00 澎湖跨海大橋
- [ ] 11:15 - 12:00 小門鯨魚洞
- [ ] 12:10 - 13:10 午餐：自備蛋奶素便當、麵包或三明治
- [ ] 13:30 - 14:00 大菓葉柱狀玄武岩
- [ ] 14:20 - 15:20 二崁聚落
- [ ] 15:40 - 16:30 西嶼西臺或漁翁島燈塔
- [ ] 17:30 - 19:00 回馬公晚餐：蔡記饡蔬食坊或市區其他蔬食店

**備註**
- 北環沿線素食較少，建議先在馬公備餐
- 蛋奶素這天可多準備麵包、飯糰、起司三明治、優格、豆漿
- 北環與西嶼多數景點開車方便，通常也比馬公市區好停車

---

## Day 3｜七美＋望安跳島

- [ ] 07:30 - 08:00 早餐：市區先買好
- [ ] 08:00 - 08:30 前往南海遊客中心報到
- [ ] 08:30 - 16:30 七美＋望安跳島一日遊
- [ ] 09:30 - 12:00 七美：雙心石滬、小台灣等景點
- [ ] 12:00 - 13:00 午餐：自備蛋奶素餐盒
- [ ] 13:30 - 15:30 望安：花宅聚落、島上海景
- [ ] 16:30 - 17:00 返回馬公
- [ ] 18:00 - 20:00 晚餐：澎福素食／澎湖煎餅 或回市區找蔬食餐廳

**備註**
- 跳島日建議自備午餐
- 蛋奶素可準備飯糰、蛋沙拉三明治、麵包、水果、優格、豆漿
- 開車前往南海遊客中心時，建議提早到，先預留停車與報到時間

---

## Day 4｜湖西線＋返程

- [ ] 08:00 - 09:00 早餐＋退房
- [ ] 09:30 - 10:30 奎壁山摩西分海
- [ ] 10:45 - 11:30 南寮古厝
- [ ] 11:45 - 12:45 午餐：回馬公吃蔡記饡蔬食坊或市區其他蔬食店
- [ ] 13:00 - 14:00 林投公園
- [ ] 14:10 - 15:00 隘門沙灘
- [ ] 15:00 - 16:00 買伴手禮
- [ ] 16:00 - 17:00 前往機場、還車、返程

**備註**
- 奎壁山摩西分海一定要先查潮汐時間
- 若班機較早，Day 4 可刪減部分景點
- 湖西線開車順路，景點停留可彈性調整

---

## 蛋奶素店名單

- 蔡記饡蔬食坊
- 建國炸粿
- 舒心食光
- 澎福素食／澎湖煎餅

---

## 行前提醒

- [ ] 訂好機票或船票
- [ ] 預約住宿
- [ ] 預約租車
- [ ] 預約 Day 3 跳島行程
- [ ] 查 Day 4 奎壁山潮汐時間
- [ ] 確認店家是否有蛋奶素品項
- [ ] 準備防曬、帽子、墨鏡
- [ ] 準備暈船藥
- [ ] 北環日、跳島日先備好蛋奶素餐點
- [ ] 馬公市區景點以外圍停車再步行為主
- [ ] 若碰到花火節或試放場，晚上提早出門找車位
