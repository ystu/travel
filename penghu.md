# 澎湖四天三夜行程

> 地點：澎湖  
> 天數：四天三夜  
> 住宿：馬公市區連住 3 晚  
> 住宿位置建議：馬公市區外圍，開車到中央老街、南海遊客中心約 5 到 10 分鐘  
> 交通：自駕開車  
> 飲食：蛋奶素  

---

## Day 1｜馬公市區＋南環海景

- [ ] 07:50 抵達馬公
- [ ] 08:10 - 09:00 租車、前往住宿寄放行李
- [ ] 09:00 - 10:00 早餐或早午餐：市區先吃
- [ ] 10:15 - 11:15 篤行十村
- [ ] 11:30 - 12:30 中央老街、天后宮
- [ ] 12:40 - 13:40 午餐：蔡記饡蔬食坊
- [ ] 14:00 - 15:00 觀音亭、彩虹橋
- [ ] 15:30 - 16:30 風櫃洞
- [ ] 16:45 - 17:30 山水沙灘
- [ ] 18:00 之後 晚餐：回市區找蔬食餐廳

**備註**
- 第一天抵達時間早，先走馬公市區再往南環會比較順
- 蛋奶素可優先選早餐店、咖啡店、蔬食店的蛋奶品項
- 馬公市區晚上較熱鬧，建議停外圍再步行進中央老街、天后宮一帶

---

## Day 2｜北環一日遊

- [ ] 08:00 - 09:00 早餐：舒心食光或先買早餐
- [ ] 09:15 - 09:40 中屯風車
- [ ] 10:00 - 10:30 通梁古榕
- [ ] 10:40 - 11:00 澎湖跨海大橋
- [ ] 11:15 - 12:00 小門鯨魚洞
- [ ] 12:10 - 13:10 午餐：自備蛋奶素便當、麵包或三明治
- [ ] 13:30 - 14:00 大菓葉柱狀玄武岩
- [ ] 14:20 - 15:20 二崁聚落
- [ ] 15:40 - 16:30 西嶼西臺或漁翁島燈塔
- [ ] 17:30 - 19:00 回馬公晚餐：蔡記饡蔬食坊或市區其他蔬食店

**備註**
- 北環沿線素食較少，建議先在馬公備餐
- 蛋奶素這天可多準備麵包、飯糰、起司三明治、優格、豆漿
- 北環與西嶼多數景點開車方便，通常也比馬公市區好停車

---

## Day 3｜湖西＋東衛慢遊

- [ ] 08:00 - 09:00 早餐：市區先吃或買好帶著
- [ ] 09:30 - 10:30 奎壁山摩西分海
- [ ] 10:45 - 11:30 南寮古厝
- [ ] 11:45 - 12:45 午餐：回馬公或湖西周邊簡單吃
- [ ] 13:15 - 14:15 龍門閉鎖陣地文化園區
- [ ] 14:30 - 15:00 菓葉灰窯
- [ ] 15:15 - 16:00 青螺濕地
- [ ] 16:30 - 17:30 澎湖海洋地質公園中心或澎湖生活博物館
- [ ] 18:30 - 20:00 晚餐：澎福素食／澎湖煎餅 或回市區找蔬食餐廳

**備註**
- 這天改成完全不跳島，行程更穩也更輕鬆
- 奎壁山摩西分海一定要配合潮汐時間，建議前一天先查
- 湖西沿線這樣排比較不會和 Day 4 重複，整體更順

---

## Day 4｜機場順路景點＋返程

- [ ] 08:00 - 09:00 早餐＋退房
- [ ] 09:00 - 09:40 買伴手禮
- [ ] 09:50 - 10:30 林投公園或隘門沙灘二選一
- [ ] 10:30 - 11:00 前往機場方向
- [ ] 11:00 - 12:00 還車、辦理報到
- [ ] 13:00 班機起飛

**備註**
- 下午 1 點班機建議最晚 11 點前到機場
- 最後一天不要排市區午餐，避免時間太緊
- 若想更從容，Day 4 可只留買伴手禮後直接去機場

## 行前提醒

- [ ] 訂好機票或船票
- [ ] 預約住宿
- [ ] 預約租車
- [ ] 查 Day 3 或 Day 4 奎壁山潮汐時間
- [ ] 確認店家是否有蛋奶素品項
- [ ] 準備防曬、帽子、墨鏡
- [ ] 北環日先備好蛋奶素餐點
- [ ] 馬公市區景點以外圍停車再步行為主
- [ ] 若碰到花火節或試放場，晚上提早出門找車位
