# 澎湖四天三夜行程

> 地點：澎湖  
> 天數：四天三夜  
> 住宿：海悅飯店連住 3 晚  
> 住宿位置：馬公市區，距澎湖機場約 15 到 20 分鐘車程，適合作為南環、北環、湖西路線的出發點  
> 交通：自駕開車  
> 飲食：蛋奶素  

## 天氣

- `4/22（三）`：約 `27/23°C`，晴到多雲，局部短暫陣雨機率偏高
- `4/23（四）`：約 `27/23°C`，雲量較多，早上到下午都有可能下點雨
- `4/24（五）`：約 `28/24°C`，大致晴朗，適合安排海邊或跳島行程
- `4/25（六）`：約 `24/19°C`，轉涼，偏陰，局部短暫雨
- `4/26（日）`：約 `24/21°C`，晴到多雲，降雨機率下降

> 備註：`4/22 - 4/23` 較悶、可能有陣雨；`4/24` 天氣相對最好；`4/25` 轉涼，風感可能比較明顯。

## Day 1｜馬公市區＋南環海景

- [ ] 07:50 抵達馬公
- [ ] 08:10 - 09:00 租車、前往住宿寄放行李
- [ ] 09:00 - 10:00 早餐或早午餐：市區先吃
- [ ] 10:15 - 11:15 篤行十村
- [ ] 11:30 - 12:30 中央老街、天后宮
- [ ] 12:40 - 13:40 午餐：蔡記饡蔬食坊
- [ ] 14:00 - 15:00 觀音亭、彩虹橋
- [ ] 15:30 - 16:30 風櫃洞
- [ ] 16:45 - 17:30 山水沙灘
- [ ] 18:00 之後 晚餐：回市區找蔬食餐廳

**備註**
- 第一天抵達時間早，先走馬公市區再往南環會比較順
- 蛋奶素可優先選早餐店、咖啡店、蔬食店的蛋奶品項
- 馬公市區晚上較熱鬧，建議停外圍再步行進中央老街、天后宮一帶

## Day 2｜北環一日遊

- [ ] 08:00 - 09:00 早餐：舒心食光或先買早餐
- [ ] 09:15 - 09:40 中屯風車
- [ ] 10:00 - 10:30 通梁古榕
- [ ] 10:40 - 11:00 澎湖跨海大橋
- [ ] 11:15 - 12:00 小門鯨魚洞
- [ ] 12:10 - 13:10 午餐：自備蛋奶素便當、麵包或三明治
- [ ] 13:30 - 14:00 大菓葉柱狀玄武岩
- [ ] 14:20 - 15:20 二崁聚落
- [ ] 15:40 - 16:30 西嶼西臺或漁翁島燈塔
- [ ] 17:30 - 19:00 回馬公晚餐：蔡記饡蔬食坊或市區其他蔬食店

**備註**
- 北環沿線素食較少，建議先在馬公備餐
- 蛋奶素這天可多準備麵包、飯糰、起司三明治、優格、豆漿
- 北環與西嶼多數景點開車方便，通常也比馬公市區好停車
- 西嶼西臺和漁翁島燈塔建議二選一，不要兩個都硬塞；想看夕陽再優先選漁翁島燈塔

## Day 3｜湖西＋東衛慢遊

- [ ] 08:00 - 09:00 早餐：市區先吃或買好帶著
- [ ] 09:30 - 10:30 奎壁山摩西分海
- [ ] 10:45 - 11:30 南寮古厝
- [ ] 11:45 - 12:45 午餐：回馬公或湖西周邊簡單吃
- [ ] 13:15 - 14:15 龍門閉鎖陣地文化園區
- [ ] 14:30 - 15:00 菓葉灰窯
- [ ] 15:15 - 16:00 青螺濕地
- [ ] 16:30 - 17:30 澎湖海洋地質公園中心或澎湖生活博物館
- [ ] 18:30 - 20:00 晚餐：澎福素食／澎湖煎餅 或回市區找蔬食餐廳

**備註**
- 這天改成完全不跳島，行程更穩也更輕鬆
- 奎壁山摩西分海一定要配合潮汐時間，建議前一天先查；若潮汐不在上午，就把奎壁山和後面的湖西景點順序互換
- 龍門閉鎖陣地文化園區建議先確認是否需要預約或配合導覽時段，避免到了不能入場
- 湖西沿線這樣排比較不會和 Day 4 重複，整體更順

## Day 4｜機場順路景點＋返程

- [ ] 08:00 - 09:00 早餐＋退房
- [ ] 09:00 - 09:40 買伴手禮
- [ ] 09:50 - 10:30 林投公園或隘門沙灘二選一
- [ ] 10:30 - 11:00 前往機場方向
- [ ] 11:30 - 12:30 還車、辦理報到
- [ ] 13:50 班機起飛

**備註**
- 13:50 班機建議最晚 11:30 到 12:00 之間抵達機場或還車點
- 最後一天不要排市區午餐，避免時間太緊
- 伴手禮若在馬公市區買，買完就直接往林投、隘門、機場方向走，不要再折回市區
- 海悅飯店到澎湖機場車程約 15 到 20 分鐘，最後一天仍建議預留還車、加油和報到緩衝
- 若想更從容，Day 4 可只留買伴手禮後直接去機場

## 景點簡介

- **篤行十村**：澎湖很有代表性的眷村聚落，保留老屋、巷弄和文創小店，也和張雨生、潘安邦的故事有連結。適合慢慢散步、拍照，作為抵達後的輕鬆第一站很剛好。
- **中央老街**：馬公市區最有老城味的一段街區，周邊有古井、老屋、伴手禮店和小吃。街道不長，但很適合和天后宮一起排，順路感受馬公早期街市氛圍。
- **天后宮**：台灣歷史很久的媽祖廟之一，也是馬公老城區的核心地標。建築細節和廟埕氣氛都值得看，和中央老街距離很近。
- **觀音亭、彩虹橋**：觀音亭是馬公市區看海、散步和看夕陽的經典點，旁邊彩虹橋晚上點燈後很有辨識度。若遇到花火節，這一帶通常會更熱鬧，也要更早找停車。
- **風櫃洞**：澎湖南環的海蝕地形景點，海浪打進岩縫時會有像風箱一樣的聲響。這裡看的是海岸線和地質感，風大時特別有澎湖味。
- **山水沙灘**：澎湖本島很受歡迎的沙灘，沙質細、海面開闊，適合散步、看海和拍照。傍晚來比較舒服，也能避開白天最曬的時間。
- **中屯風車**：北環路線上很順路的風車景點，大片風機配海景很有澎湖的開闊感。停留時間不用太長，適合作為北環的第一個暖身點。
- **通梁古榕**：以大榕樹氣根形成的天然棚架聞名，樹蔭下很涼，也很有地方生活感。它就在通往跨海大橋的路線附近，停一下很順。
- **澎湖跨海大橋**：連接白沙和西嶼的重要地標，也是北環一定會經過的經典拍照點。橋邊風通常很大，拍照時注意帽子和隨身物品。
- **小門鯨魚洞**：因海蝕拱門外形像鯨魚而得名，是小門嶼很具代表性的地質景觀。附近也能看海岸岩層，和北環、西嶼路線搭配很自然。
- **大菓葉柱狀玄武岩**：澎湖很經典的柱狀玄武岩景點，岩壁線條整齊又壯觀，雨後或有水面倒影時特別好看。停留時間約半小時到一小時就很足夠。
- **二崁聚落**：西嶼保存良好的傳統聚落，有閩南式老宅、咾咕石牆和小巷風景。節奏比市區慢，很適合下午散步、買點小東西或坐下喝飲料。
- **西嶼西臺**：清代砲臺遺跡，能看到軍事建築和海防視角，空間感很有歷史味。若你對古蹟和軍事地景有興趣，這點會比單純拍照型景點更耐看。
- **漁翁島燈塔**：位在西嶼西端，是很適合看海和夕陽的燈塔景點。若北環當天想收在一個漂亮的遠景，這裡會比多塞景點更舒服。
- **奎壁山摩西分海**：退潮時會出現通往赤嶼的潮間帶步道，是湖西最有記憶點的景觀之一。這個點一定要配合潮汐，不然看到的感覺會差很多。
- **南寮古厝**：湖西一帶的傳統聚落，能看見澎湖老房子、農村生活和社區營造的痕跡。它不是刺激型景點，適合用慢遊心情看老屋和巷弄。
- **龍門閉鎖陣地文化園區**：由軍事設施轉型的文化園區，可以看到澎湖過去軍事防禦的空間樣貌。這類點比較吃導覽或開放狀態，出發前確認一下會更穩。
- **菓葉灰窯**：以前燒製石灰的產業遺跡，規模不大，但能補上澎湖地方產業史的一角。它適合和湖西線其他點順遊，不建議單獨特地繞很遠。
- **青螺濕地**：看潮間帶、濕地和水鳥生態的地方，景色安靜，和沙灘、玄武岩的風景不太一樣。天氣好時視野很開，適合放慢速度停一下。
- **澎湖海洋地質公園中心**：適合用室內方式快速理解澎湖的玄武岩、海蝕地形和地質背景。若前幾天看了很多岩石和海岸，來這裡會比較知道自己看到了什麼。
- **澎湖生活博物館**：用展覽方式介紹澎湖的日常生活、信仰、漁業和地方文化。若想把旅程從「看景點」補成「理解澎湖」，這裡很適合。
- **林投公園**：靠近機場方向的海邊公園，樹蔭和海岸步道讓最後一天不會太趕。適合退房後、還車前短暫散步。
- **隘門沙灘**：距離機場近的沙灘，海景開闊，也很適合排在返程前輕鬆收尾。若時間有限，和林投公園二選一即可。
- **東衛石雕公園**：以石雕作品和戶外散步空間為主，屬於可替換的輕量景點。若想在市區到湖西之間補一個不用太久的點，可以考慮放進去。
